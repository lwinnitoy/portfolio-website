    Hi, I'm Liam Winnitoy

    I Build Data Systems

    I'm a Computer Science student at the University of Victoria with a passion for building systems that handle data with clarity and purpose. I enjoy turning raw information into useful tools.

My first project was a simple web application that allows users to input their favorite movies and receive recommendations based on their preferences. I used HTML, CSS, and JavaScript to create the front-end interface, and Node.js for the back-end server. The project helped me understand the basics of web development and how to work with APIs.

My second project was a data visualization tool that allows users to explore large datasets using interactive charts and graphs. I used D3.js for the front-end and Python with Flask for the back-end. This project taught me how to handle large amounts of data and create meaningful visualizations.

My third project was a machine learning application that predicts house prices based on various features. I used Python with Scikit-learn for the machine learning model and Flask for the web application. This project helped me understand the basics of machine learning and how to deploy models in a web application.
