    Hi, I'm Liam Winnitoy

    I Build Data Systems

    I'm a Computer Science student at the University of Victoria with a passion for building systems that handle data with clarity and purpose. I enjoy turning raw information into useful tools.

My first project was a simple web application that allows users to input ingredients and receive recipes based on their food choice. I used HTML, CSS, and JavaScript to create the front-end interface, and spoonacular.com as the API. The project helped me understand the basics of web development and how to work with APIs.

My second project was a medical clinic patient system. It allows users to mangage patient records, information, and appointments. I used Python with pickle and JSON for persistence, and PyQT6 for the UI. This project taught me how to utalize CRUD operations and DAOs to manage databases.

My third project was a solution for a flight allocation problem using Edmonds-Karp network flow algorithm. I used Java to write a graph, edge, and pair class. I learned how to implement a graph and network flow algorithm and used them to solve real-world problems.
