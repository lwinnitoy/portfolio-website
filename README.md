# Portfolio Website - Liam Winnitoy

A professional portfolio website showcasing my software engineering and data engineering projects. Built with semantic HTML5, modern CSS, and vanilla JavaScript.

🔗 **Live Site:** [https://lwinnitoy.github.io/portfolio-website/](https://lwinnitoy.github.io/portfolio-website/)

## Features

- ✨ Clean, modern dark theme design
- 📱 Fully responsive (desktop, tablet, mobile)
- ♿ Accessible with ARIA labels and semantic HTML
- 🎯 SEO optimized with meta tags and Open Graph
- 📊 Animated scroll progress indicator
- 🎨 Smooth animations and transitions
- 📋 Modular project card system for easy updates

## Sections

1. **Hero** - Introduction and personal branding
2. **About** - Background and career goals
3. **Skills** - Technical skills organized by category
4. **Education** - Academic background
5. **Projects** - Showcase of software projects with detailed descriptions
6. **Contact** - Professional contact information

## Tech Stack

- **HTML5** - Semantic markup
- **CSS3** - Modern styling with CSS variables, flexbox, and grid
- **JavaScript (ES6+)** - Smooth scrolling and progress indicator
- **Roboto Font** - Professional typography

## Adding New Projects

See [PROJECT_TEMPLATE.md](PROJECT_TEMPLATE.md) for a complete guide on adding new projects to your portfolio.

Quick steps:
1. Add your project image to `images/` folder
2. Copy the project card HTML template
3. Customize with your project details
4. Add tech stack tags and highlights

## Development

To run locally:
1. Clone the repository
2. Open `index.html` in a web browser
3. No build process required!

## Deployment

This site is deployed using GitHub Pages:
1. Push changes to the repository
2. GitHub Pages automatically serves from the repository
3. View live at: https://lwinnitoy.github.io/portfolio-website/
